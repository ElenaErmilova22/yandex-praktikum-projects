# Разработка алгоритма для защиты данных

## Цель
Защитить данные клиентов страховой компании, чтобы при преобразовании качество моделей машинного обучения не ухудшилось. Обосновать преобразования.
## Вывод
С помощью умножения на обратимую матрицу P мы смогли зашифровать данные пользователей, что не повлияло на качество модели. Также умножение на обратную матрицу P позволяет восстановить исходные данные.
## Используемые инструменты
Pandas, sklearn, numpy, Seaborn, машинное обучение
